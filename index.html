Builderius CSS framework

Builderius CSS is a utility-first framework designed to simplify and speed up web development. It provides a mix of atomic classes and pre-styled components, allowing for flexible and consistent design. The framework includes basic UI elements like buttons (filled, tonal, outlined, and ghost), badges, form fields, and accordions. Additionally, it offers custom components such as pagination and a modal-based cookie notice dialog. Built with scalability in mind, Builderius CSS helps developers create responsive and maintainable interfaces with ease.
